## Terraform Fundamentals Cheat Sheet

```bash
# Download the Provider
terraform init

# Automatically Format the Syntax to the Canonical Standard Style
terraform fmt

# Validate & Check the Code
terraform validate

# Generate Plan to Build the Infra
terraform plan

# Applying the Plan
terraform apply

# Applying the Plan w Auto Approve
terraform apply -auto-approve

# Show Current State of Infra
terraform show

# Destroy the Infra
terraform destroy
```
